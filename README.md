digits
===========
![homepage](https://raw.github.com/evashek/digits/Attempt-1/doc/frontpage.png)

Overview
===========
A database of contact information.

Installation
===========
Download the zip file, unzip, cd into directory, invoke the Play console, and execute the run command to see the application at http://localhost:9000.

Usage
===========
Look up contacts on the front page.  To add a contact click the "New Contact" link in the navbar and fill in the form.
